### REST API use Express.js and MySQL
### Usage
### GET USERS (GET)
```js
    http://localhost:4000/api/users
```
###
### GET USER (GET)

```js
    http://localhost:4000/api/users/id
```
###
### CREATE USER (POST)
```js
    http://localhost:4000/api/users
```
###
### UPDATE USER (PUT)

```js
    http://localhost:4000/api/users/id
```
###
### DELETE USER (DELETE)

```js
    http://localhost:4000/api/users/id
```
